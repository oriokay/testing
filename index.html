Connect Google Sheets
Every order will be saved automatically to your spreadsheet. Setup takes about 3 minutes and is completely free.
  1. Go to sheets.google.com → Create a new blank spreadsheet
  2. In Row 1, add these headers: Timestamp | Order ID | Name | Email | Phone | Address | City | Items | Total | Notes
  3. Go to Extensions → Apps Script and replace all code with:
function doPost(e) { var sheet = SpreadsheetApp .getActiveSpreadsheet().getActiveSheet(); var data = JSON.parse(e.postData.contents); sheet.appendRow([ data.timestamp, data.orderId, data.name, data.email, data.phone, data.address, data.city, data.items, data.total, data.notes ]); return ContentService .createTextOutput(JSON.stringify({status:"ok"})) .setMimeType(ContentService.MimeType.JSON); }
